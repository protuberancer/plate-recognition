# plate-recognition

В ходе практики была поставлена задача по распознаванию автомобильных номеров с использованием технологий OCR. Экспериментальная часть показала, что применение модели paddleocr позволяет достичь точности распознавания порядка 90% на датасете, где номера представлены крупным планом и без посторонних объектов. Однако, результаты работы системы на реальных фотографиях, содержащих дополнительный текст, оказались неудовлетворительными, что указывает на необходимость доработки алгоритма. Ограничения аппаратного и программного обеспечения стали причиной невозможности дообучения моделей, что в будущем можно компенсировать использованием подходящего ПК или облачных сервисов. Полученный опыт демонстрирует потенциал OCR технологий для решения подобных задач, а также определяет направление дальнейших исследований: разработку методов выделения именно автомобильных номеров для повышения точности распознавания и внедрение дополнительных этапов предварительной обработки изображений.
